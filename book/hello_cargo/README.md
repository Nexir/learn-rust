# Hello, Cargo !

## Create new project (crate)

```bash
cargo new hello_cargo
```

## Build project

```bash
cargo build
```

Dev binaries location

```bash
./target/debug/hello_cargo
```

## Run project

Builds and run project

```bash
cargo run
```

## Check project

Check if project compiles without producing executable (faster than building)

```bash
cargo check
```
